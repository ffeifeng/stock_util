# 股票选股脚本说明

## 文件清单

### 1. 选股-起爆点分析.py（主力用）
**策略：主力建仓 + 缩量洗盘 + 温和突破 MA30**

筛选条件：
- 主力建仓阶段：区间内最高点相对最低点涨幅 ≥ 20%
- 洗盘阶段：20~90 个交易日（1~3个月），成交量明显萎缩（缩量比 < 0.8）
- 突破条件：收盘由下往上穿越 30 日均线，突破幅度 0.5%~12%
- 价格过滤：股价 ≥ 3 元，排除 ST、北交所

评分加分项：
- **强势型**（跌破MA30仅1天立刻拉回）：+8分
- **守住突破价**（突破后从未跌回突破日收盘）：+10分

**用法：**
```bash
# 扫描最近5个交易日（默认）
python 选股-起爆点分析.py

# 扫描指定日期范围
python 选股-起爆点分析.py 20260301 20260312
```
输出文件：`D:\soft\选股结果-起爆点.json`

### 2. 选股-突破30均线.py
**策略：简单版突破 30 日均线筛选**

扫描 2026年1月1日~1月10日 内突破30均线的股票，参照样本：浙江医药(002826)形态。

```bash
python 选股-突破30均线.py
```

---

### 3. 选股-突破30均线V2.py
**策略：精准版突破 30 日均线（更严格）**

在V1基础上增加：
1. 突破前成交量快速放大（≥前5日均量50%）
2. 突破幅度 1%~8%（不能太小也不能太大）
3. 股价 ≥ 3 元

```bash
python 选股-突破30均线V2.py
```

---

### 4. 选股-基量能洗盘突破.py（V3）
**策略：主力建仓 + 长期洗盘 + 突破动态验证**

动态条件：
1. 突破前3~6月内有一段上涨（涨幅 ≥ 20%）
2. 上涨后进入长期横盘洗盘（60天以上）
3. 洗盘期间成交量逐渐萎缩，振幅平稳
4. 最后温和突破30均线（不要求量能）

```bash
python 选股-基量能洗盘突破.py
```

## 结果字段说明

| 字段 | 含义 |
|------|------|
| score | 综合评分（越高越好）|
| pattern_type | 强势型(strong) / 稳健型(stable) |
| hold_breakout_close | 突破后从未跌破突破日收盘价（主力高控盘信号）|
| break_date | 突破日期 |
| break_close | 突破日收盘价 |
| break_pct | 突破幅度(%) |
| pullback_date | 突破后最低回踩日 |
| restable_date | 回踩后再次企稳日（重回突破价以上）|
| surge_pct | 主力建仓阶段涨幅(%) |
| shrink_ratio | 洗盘期相对建仓期的缩量比（越小越好）|
| wash_days | 洗盘天数 |
| break_vol_ratio | 突破日成交量/前10日均量 |
